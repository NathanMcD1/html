Destiny 2 Strand Warlock Build

*This code is dedicated to my current warlock build for the game Destiny 2*

Build Summary:

The main focus of this build is to focus on ad control by spawning threadlings and tangles to take out eneimes in groups. In this build, I use the Quicksilver Storm to generate tangles using it's grenade launcher. Plus, since it's a strand weapon, you can use it with the fragment: Thread of Rebirth and any harmonizing armor mods I also make use of the Swarmers to generate threadlings when using/destroying the tangles. Overral, this is a basic build, but it is very fun to play around with.
